# CRYPTOCURRENCY STATS APP

## 💻 Techs
- React.js
- Context API
- Axios
- React Icons
## 📦 Installation

### 📝 Clone the project

```bash
git clone https://github.com/eyupucmaz/crypto-app.git
```
### ✅ Install Dependencies
With yarn:
```bash
cd ./crypto-app && yarn
```
With npm:
```bash
cd ./crypto-app && npm install
```
### 🏃‍♂️ Run the project
With yarn:
```bash
yarn start
```
With npm:
```
npm run start
```
